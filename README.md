# 学生の名前と点数を管理するプログラム
### 名前の追加、削除、点数の更新、平均点の計算ができるプログラムです。

```
入力例:
    1. 学生を追加
    2. 学生を削除
    3. 点数を更新
    4. 平均点を計算
    5. 全学生の情報を表示
    6. 終了
    数字を選択してください: 1

    追加する学生の名前を入力してください: 山田太郎
    山田太郎の点数を入力してください: 85
    追加が完了しました。

    1. 学生を追加
    2. 学生を削除
    3. 点数を更新
    4. 平均点を計算
    5. 全学生の情報を表示
    6. 終了
    数字を選択してください: 1

    追加する学生の名前を入力してください: 鈴木花子
    鈴木花子の点数を入力してください: 92
    追加が完了しました。

期待される出力例:
    1. 学生を追加
    2. 学生を削除
    3. 点数を更新
    4. 平均点を計算
    5. 全学生の情報を表示
    6. 終了
    数字を選択してください: 5

    学生一覧:
    山田太郎: 85点
    鈴木花子: 92点

    1. 学生を追加
    2. 学生を削除
    3. 点数を更新
    4. 平均点を計算
    5. 全学生の情報を表示
    6. 終了
    数字を選択してください: 4

    平均点: 88.5 点
    
    1. 学生を追加
    2. 学生を削除
    3. 点数を更新
    4. 平均点を計算
    5. 全学生の情報を表示
    6. 終了
    数字を選択してください: 3
    
    点数を更新する学生の名前を入力してください: 山田太郎
    山田太郎の点数を入力してください: 88
    更新が完了しました。
    
    1. 学生を追加
    2. 学生を削除
    3. 点数を更新
    4. 平均点を計算
    5. 全学生の情報を表示
    6. 終了
    数字を選択してください: 2

    削除する学生の名前を入力してください: 鈴木花子
    鈴木花子の削除が完了しました。
    
    1. 学生を追加
    2. 学生を削除
    3. 点数を更新
    4. 平均点を計算
    5. 全学生の情報を表示
    6. 終了
    数字を選択してください: 5

    学生一覧:
    山田太郎: 88点

    1. 学生を追加
    2. 学生を削除
    3. 点数を更新
    4. 平均点を計算
    5. 全学生の情報を表示
    6. 終了
    数字を選択してください: 6

    プログラムを終了します。
```
